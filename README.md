### Requirements
AstroNvim@^2.0.0
neovim@^0.8

### Clone the repository


```
git clone https://github.com/AstroNvim/AstroNvim ~/.config/nvim
nvim +PackerSync
cd ~/.config/nvim
git clone https://github.com/codejunkienick/astronvim_userconfig ./user
nvim +PackerSync
```
